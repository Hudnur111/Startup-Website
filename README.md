## ⚙️ Tech-Stack

Statisches HTML/CSS/JS — kein Build-Prozess, kein Framework. Formulare laufen über **Netlify Forms**.

## 🚀 Lokal ansehen

```bash
# Einfach im Browser öffnen
open index.html
```

## ☁️ Deployment

Automatisches Deployment über Netlify bei jedem Push. Konfiguration: [`netlify.toml`](netlify.toml)
